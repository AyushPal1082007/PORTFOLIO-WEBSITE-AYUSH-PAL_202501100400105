Portfolio Website
A modern, responsive personal portfolio website showcasing my projects, skills, and contact information.
Overview
This is a single-page portfolio website built with HTML and CSS, featuring a modern design with animated gradients, smooth scrolling, and responsive layout. The site includes sections for introduction, about, skills, projects, and contact information.
Features
Modern Design: Vibrant gradient background with floating orb animations
Smooth Navigation: Fixed header with smooth scroll to sections
Responsive Layout: Adapts to different screen sizes (desktop, tablet, mobile)
Interactive Elements: Hover effects on buttons, cards, and navigation links
Project Showcase: Grid layout displaying projects with images and descriptions
Contact Section: Email and social media links (GitHub, LinkedIn)

Technologies Used
HTML5
CSS3
Pure CSS animations (no JavaScript frameworks)
Sections

Home: Introduction with profile photo and call-to-action buttons
About: Brief description of background and goals
Skills: List of technical skills (C, Python, HTML & CSS)
Projects: Showcase of completed projects with preview links
Contact: Email address and social media links
